# Reference Repos

These repositories are vendored for comparison and learning only. They are not part of the build or deployment workflow for the Aztec port.

## Contents
- `darkforest-v0.6`: Canonical v0.6 contracts, circuits, and client.
- `darkforest-local`: Runnable local Dark Forest for behavior reference.
- `aztec-starter`: Aztec/Noir starter repo reference.
- `noir`: Noir language/toolchain reference.

## Notes
- Do not import code directly from `reference/` into `packages/` or `apps/` without an explicit port.
